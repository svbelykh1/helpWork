Cоздание Виртуальной машины на примере VirtualBox
  1.Усьановка VirtualBox 
    1.1 Переходим на сайт https://www.virtualbox.org/
    1.2 Downloads->для Windows->Windows hosts
  2.Скачиваем образ диска ОС, которую хотим установить
    2.1 переходим на https://ubuntu.com/
    2.2 Downloads->UbuntuDesktop->скачиваем
    2.3 Можно использовать альтернативное скачивание(например через торрент)
  3.Создаем VM 
   3.1 Жмякаем создать-->Вводим Имя->Выбрать тип -->Версия-->Далее
   3.2 Выделяем память (3072) она откусывается от основной ОП системы-->Далее
   3.3 Жесткий Диск(Оставляем все, как есть) Устанавливается Виртуальный жесткий жиск на который будет устанавливаться система -->Далее

   3.4 Тип ЖД для большинства подходит VDI(Virtual Disk Image)-->Next
   3.5 Выбираем динамический ЖД-->Далее
   3.6 Память для ВиртЖестДиска(20Гб)-->Создать
   
  4.Создали ВМ
    4.1 Нажимаем настроить-->Система-->процессор устанваливаем 2 ядра
    4.2 Дисплей-->учтанавливаем 128МБ --> У меня и у оратора это максимальное
    4.3 Носители-->Видим 2 насителя ВЖД Ubuntu.vdi реализован как обычный файл и оптичесий носитель Виртуаоьный дисковод, к нему и подключим наш образ который скачивали с ОС  и нажимаем ОК
   5.Запуск ВМ
    5.1 Нажимаем Запуск(начался процесс учтановки)
    5.2 svbelykh(Qq123456)
  6.Устанавливаем Guest VBoxGuestAdditions - это улучшит интергацию с гостевой системой(VBoxGuestAdditions.iso) образ находится в папке, где установлена virtualbox(C:\Program Files\Oracle\VirtualBox)
   6.1 Для начала установим пакет sudo apt install dkms build-essential
   6.2 При подключении образа возникает ошибка (Could not mount the media/drive 'C:\Program Files\Oracle\VirtualBox\VBoxGuestAdditions.iso' (VERR_PDM_MEDIA_LOCKED))
       Возможно образ уже примонтирован, если нет то выключаем ВМ убираем, что примонтировано и пробуем примонтировать в настройки -->Носители 
   6.3 Если примонтировали, нужно запустить файл VboxLinuxAdditions.run, он находится media/username/образ.. выполняем команду sudo bash VboxLinuxAdditions.run-->(перезапускаем)
   6.4 После, как установили гостевые дополнения, выключим машину и настроим общие папки
   6.5 ВМ-->Настроить-->Общие папки-->Создаем папку-->ок
   6.6 Включаем машину видим папку-->нужно добавить пользователя в груупу sudo adduser svbelykh vboxsf-->перезапускаем-->ЭВРИКА
